# JTCSV

Deep convert JSON blob to CSV.

- [Getting Started](#getting-started)
- [Todo](#todo)
	- [Features](#features)

## Getting Started

```
$ git clone https://github.com/qcom/jtcsv.git
$ cd jtcsv
$ npm install
$ gulp
$ node dist/app.js
```
